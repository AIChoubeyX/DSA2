# Pattern Printing - DSA

Simple pattern printing programs using loops.

## Programs

- **basic.cpp** - Basic C++ program
- **abcd.cpp** - Alphabet pattern
- **onetwothree.cpp** - Number pattern
- **onetwo.cpp** - Pattern program

## Compile & Run

```bash
g++ -o basic basic.cpp
./basic
```

Each program takes input for number of rows and prints patterns accordingly.
