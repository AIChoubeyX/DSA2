# DSA2 - Data Structures and Algorithms Practice

A collection of C++ programs covering fundamental programming concepts, operators, loops, and pattern printing exercises.

### 1. Operators (`operators/`)
Programs demonstrating various C++ operators:
- **comparison.cpp** - Comparison operators (==, !=, >, <, >=, <=)
- **logical.cpp** - Logical operators (&&, ||, !) with vowel/consonant checker
- **main.cpp** - Increment/Decrement operators demonstration (++, --)

### 2. While Loops (`whileloop/`)
Loop control structures:
- **basic.cpp** - Do-while loop printing multiplication table (1 to 10)

### 3. Pattern Printing (`patternprinting/`)
Various pattern printing programs using nested loops:
- **basic.cpp** - Basic square patterns
- **abcd.cpp** - Alphabet patterns (a b c d...)
- **onetwothree.cpp** - Sequential number grid (1 2 3 4...)
- **onetwo.cpp** - Number patterns
- **tri.cpp** - Triangle patterns with stars, numbers, and characters
- **hardtriangle.cpp** - Right-aligned triangle with numbers (1 2 3...)
- **pallindromepryamid.cpp** - Palindrome pyramid pattern (1 2 1, 1 2 3 2 1...)
- **pp1.cpp** - Additional pattern variations
- **top5.cpp** - Top 5 pattern exercises
